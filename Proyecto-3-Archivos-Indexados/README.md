# Proyecto 3 - Actividad integradora de evaluación_C3}

## Descripción del Sistema

Aplicación desarrollada en Python para procesar información comercial almacenada en archivos CSV. El sistema analiza las ventas semestrales de una tienda de tecnología y genera reportes tabulares, visualizaciones gráficas e indicadores estratégicos para apoyar la toma de decisiones.

## Tecnologías Utilizadas

* Python 3
* Pandas
* Matplotlib

## Formato de Archivo

* CSV

## Justificación del Formato

Se seleccionó el formato CSV debido a su eficiencia para almacenar información tabular en filas y columnas. Su estructura facilita la lectura mediante Pandas y permite realizar operaciones de agrupamiento, filtrado y análisis estadístico con bajo costo computacional.

## Funcionalidades Implementadas

* Ventas totales por producto.
* Ventas totales por mes.
* Ingresos por producto.
* Producto más vendido.
* Gráfica de barras.
* Gráfica de líneas.
* Gráfica circular.
* Reporte ejecutivo para la toma de decisiones.

## Operaciones de Archivos

* Lectura mediante `pd.read_csv()`.
* Escritura de reportes mediante `open()` y `write()`.
* Procesamiento tabular con Pandas.
* Visualización de información con Matplotlib.

## Competencia Desarrollada

Implementar técnicas de visualización y representación de información mediante herramientas de programación para interpretar datos y apoyar la toma de decisiones.
