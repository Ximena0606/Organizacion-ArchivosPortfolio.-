## Objetivo del Portafolio

Este repositorio reúne los proyectos desarrollados durante el semestre en la asignatura Administración y Organización de Datos. Cada proyecto fue diseñado para aplicar diferentes técnicas de almacenamiento, lectura, escritura, organización y recuperación de información mediante archivos, permitiendo evaluar su impacto en el rendimiento y eficiencia de los sistemas informáticos.

La evidencia presentada demuestra la competencia de evaluar distintas organizaciones de archivos aplicándolas a situaciones reales, considerando aspectos como velocidad de acceso, uso de almacenamiento, flexibilidad estructural y escalabilidad.

## Conclusión General

El desarrollo de estos proyectos permitió comprender la importancia de la organización de archivos como base para el almacenamiento y recuperación eficiente de información. Asimismo, se analizaron las ventajas y limitaciones de diferentes formatos de archivo, fortaleciendo los conocimientos necesarios para abordar sistemas de gestión de bases de datos y arquitecturas de información más complejas.
